# Contributing

Thanks for wanting to contribute! Please:
- Fork the repo on GitHub
- Create a branch `feature/your-feature`
- Open a PR with tests and a short description
- Follow the style in the repo (black / flake8 / mypy if configured)

See `CODE_OF_CONDUCT.md` and `CONTRIBUTING_GUIDE.md` in the repo for more.
