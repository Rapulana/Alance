Getting Started

Welcome to the Alance Developer Kit (ADK). Follow these steps to install and initialize your SDK.

Installation

pip install alance-sdk

Initialize SDK

import alance

alance.init(api_key="your_api_key_here")

Start building your first AI or Quantum app with just a few lines of code!
