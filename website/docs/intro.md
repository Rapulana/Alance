# Welcome to Alance

Alance is an open developer kit for building quantum + AI applications. It provides:
- a hardware-agnostic runtime and executor,
- algorithm templates,
- visualization & analysis utilities,
- examples and tests for common quantum problems.

This site is the developer documentation for Alance — quickstart, API guide, examples, and contributing.
